Interactive 2D Fluid Simulation

Entranced by the beauty of turbulent smoke plumes and swirling soap bubbles, I sought to understand the physics behind fluid flow.

I worked through Lorena Barba's course 12 steps to Navier-Stokes, to write a physically accurate 2D incompressible fluid solver, directly implementing the N-S equations. Whilst it helped develop a good understanding of the equations and a simple way to implement them, the numerical scheme is still unstable for most flow variables.

Therefore I changed the solver to now implement the method presented in Jos Stam's paper Real-Time Fluid Dynamics for Games, which is unconditionally stable and can still run in real-time.

UROP: High-order mesh generation for NekMesh

As part of a 12-week research project with Imperial, I helped develop an interface between the commercial mesh generation software ANSA and the in-house high-order meshing tool NekMesh, part of the Nektar++ suite.

The interface must be able to handle high-order (up to 4th) meshes. I had to develop an algorithm for handling 'troublesome' meshes, since the particular coordinate system used doesn't allow for certain configurations.

I wrote documentation, and presented my results to end-users at RollsRoyce and developers at ANSA, summarised in this PowerPoint.
