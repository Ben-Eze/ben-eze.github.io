Real-time 2D Fluid Simulation

Check out the theory and code behind writing a real-time fluid simulation, with comparison between a couple different implementations!

Keen to understand the physics behind the beautiful swirling patterns in turbulent flow, I followed the course 12 steps to Navier-Stokes, adding using methods suggested in Jos Stam's paper Real-Time Fluid Dynamics for Games to improve stability for real-time use.

View the code on GitHub

INTERACTIVE: Predictive Shakespeare

Play around with this interactive demo, a language model trained on the entire works of Shakespeare - who knows, maybe you could write the next great classic...

Inspired by the rapid progress in LLMs such as ChatGPT and Claude, I investigated the theory behind the transformer architecture to build a character-level language model, small enough to run in real-time on a desktop!

View the code on GitHub

Research Project: High-order mesh generation

Check out the work that I did for my UROP (research project) at Imperial, working on the meshing tool NekMesh, part of the Nektar++ suite. This includes code, a presentation given to engineers at RollsRoyce and ANSA and documentation
